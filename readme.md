## ZenPen - 一个极简的写作区

Zenpen (https://imbearchild.github.io/ZenPen/) 

一个极简的写作区域。这里可以阻挡所有的干扰，并让您专注于重要的事情——写作！
（初次加载时间可能较长，请耐心等待）

## 功能

* 全屏写作，隔绝干扰
* 支持黑体、斜体、超链接与引用功能
* 可以转存为Markdown、HTML与纯文本
* 基于HTML5的自动保存


## 相对原版改进 

* 添加对中文（亚洲文字）的字数统计支持
* 添加Flotstate模式
* 添加多语言支持（中文、English）

### 使用的第三方库

* translater.js (https://github.com/jaywcjlove/translater.js)
* FontAwesome (SIL OFL 1.1 and MIT License)
